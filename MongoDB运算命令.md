# MongoDB运算命令

## 查询和映射

#### 比较查询
比较类查询有一个前提，就是比较的属性必须存在，若不存则直接判定失败。  

| 语法 | 解释 |  
| ---- | ----|  
| $eq | 等于 |  
| $ne | 不等于 |  
| $gt | 大于 |  
| $gte | 大于等于 |  
| $lt | 小于 |  
| $lte | 小于等于 |  
| $in | 等于数组中某个值 |  
| $nin | 不等于数组中任意值 |  

> 注意：  
> 对于$in和$nin来说，数组里匹配的值还可以是正则表达式，例如：db.users.find({name:{$in:[/x/,/^p/]}}） 找出名字中有字母x或以字母p为开头的人(数据)   

#### 逻辑查询

| 语法 | 解释 |  
| ---- | ---- |  
| $and | 逻辑与，数组中所有表达式必须都成立 |  
| $not | 逻辑非，数组中表达式都不成立或要对比的属性本身不存在  
| $nor | 逻辑或的取反，数组中表达式中任何一个都不成立 |  
| $or | 逻辑或，数组中表达式任何一个成立即可 |  
 

> 注意：  
> $and 执行短路查询，即若数组中前面表达式不成立，则不会继续剩余表达式的运算  
> $not 若对比属性本身不存在，运行结果也视为成立
